FAQ

Apakah ini mengambil data dari xiaomimimo.com?

Tidak. Ini proyek kreatif terinspirasi visual dan mekanik campaign token, bukan kloning resmi dan tidak memakai API pihak ketiga.

Bisa dipakai di GitHub Pages?

Bisa. Frontend murni HTML, CSS, dan JavaScript. Backend Flask tetap tersedia untuk menjalankan repo lama secara lokal.

Apa ide pengembangannya?

Tambahkan login GitHub, simpan submission ke database, dan tampilkan leaderboard kreator dengan bukti demo publik.
